Калькулятор объёма (через плечо)
Формула: лот = депозит × % / 100 ÷ цена × плечо ÷ contract_size (forex 100 000, XAU 100, XAG 5000 — из спецификации MT5). Плечо для каждого инструмента берётся из shoulders/marginal-USD.xlsx. Стоимость пункта вычисляется по правилам: quote=USD → 1×лот; base=USD → лот / цена; кросс → лот × baseUSD / цена пары. На выходе — объём и стоимость 1 пункта, чтобы вы могли решить, какой SL заложить.
